Overview

I have worked on various projects throughout my college career, starting from Winter 2022 in a reverse chronological order:

Resume Source Control App (Feb 2023 - present)

With the team of six, we have worked on this project during our capstone course in Spring 2023. Inspired by GitHub and word editors, the purpose of this project is to tailor resumes to different companies with the usage of different versions. Other features we have worked on are parsing job descriptions and seamless editing of resume blocks. I have created the design and the styling of the application.

Skills involved: ReactJS, HTML/CSS, GitHub, MySQL, Any Design, Flask, Python, spaCy

Disney Magic Kingdoms (Feb 2023 - Mar 2023)

As a Disney (and Disney Magic Kingdoms) fan, I chose to make a mock website for Disney Magic Kingdoms, and utilized responsiveness properties such as flexbox and grid to make the site optimized for all device sizes (yes, this includes mobile and 640×480 screens too)

Skills involved: ReactJS, HTML/CSS

hChat (Dec 2021 - Feb 2022)

hChat is a large-scale chat application that contains over 86,000 built-in chats divided by CUNY school, major, and course. Due to real-time chat messages, we needed a tool that handles concurrency very well.

Skills involved: ReactJS, Node/Express, Go

Two RGB Channels Off (July 2021)

This was my first project I have worked on. I have extended the functionality of Assignment #40 for the program to dynamically make multiple copies of that image that gradually disable two of the RGB channels depending on the axis.

Skills involved: Python
